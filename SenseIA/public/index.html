Perguntas frequentes


O que é o sensor DHT11?

O sensor DHT11 é um sensor de temperatura e umidade que fornece valores precisos de temperatura e umidade do ambiente.

Como funciona o sensor DHT11?

O sensor DHT11 possui um termistor para medir a temperatura e um sensor de umidade capacitivo. Ele converte esses sinais em dados digitais para leitura.

Qual a faixa de medição do sensor DHT11?

O sensor DHT11 pode medir temperaturas entre 0°C a 50°C com uma precisão de ±2°C e umidade relativa entre 20% a 90% com uma precisão de ±5%.

Quais são as aplicações comuns do sensor DHT11?

O sensor DHT11 é comumente usado em projetos de automação residencial, monitoramento ambiental, controle de umidade em estufas e em sistemas de climatização.
